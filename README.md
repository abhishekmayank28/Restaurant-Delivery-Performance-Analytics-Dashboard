# 🍽️ Restaurant & Delivery Performance Analytics Dashboard

## 📌 Project Overview

This project analyzes a large-scale food delivery dataset (120K+ records) to uncover insights into revenue trends, customer behavior, and delivery performance.

The objective is to transform raw operational data into actionable insights that support data-driven decision-making using Power BI and BigQuery.

---

## 💼 Business Problem

Food delivery platforms operate in a highly competitive environment where **customer satisfaction, delivery efficiency, and revenue growth** are critical.

However, businesses often lack clear visibility into:

* Factors affecting customer ratings
* High-performing cities and categories
* Delivery partner efficiency
* Operational delays and bottlenecks

This project addresses these challenges by building an interactive analytics dashboard for better decision-making.

---

## 🎯 Business Questions Solved

* How does delivery time impact customer ratings?
* Which categories generate the highest revenue?
* Which cities contribute most to total orders?
* Which delivery partners are most efficient?
* Where are operational delays occurring?

---

## 🛠️ Tools Used

* Power BI
* Google BigQuery
* SQL

---

## 📊 Dashboard Pages

### 1️⃣ Overview

* KPI Cards (Revenue, Orders, Rating, Delivery Time)
* Revenue Trend Analysis
* Category Performance
* Payment Method Analysis

### 2️⃣ Customer Insights

* Rating vs Delivery Time (Scatter Plot)
* Orders by Category (Stacked Bar Chart)
* Top Performing Restaurants

### 3️⃣ Operations

* Delivery Partner Performance
* Revenue by Delivery Partner
* City-wise Order Distribution
* Delivery Time Distribution

---

## 🔍 Key Insights

* Faster deliveries are associated with higher customer ratings
* A small number of categories contribute to a large share of revenue
* Certain cities consistently generate higher order volumes
* Delivery partner performance varies across regions
* Delivery delays are concentrated within specific time ranges

---

## 📈 Business Impact / Improvements

Based on the analysis:

* 🚀 Reduced delivery time can significantly improve customer satisfaction
* 📍 High-performing cities can be targeted for business expansion
* 🍽️ Top categories can be prioritized for promotions
* 🚚 Underperforming delivery partners can be optimized or replaced
* ⏱️ Bottlenecks can be addressed to improve operational efficiency

---

## ⚠️ Challenges Faced

* Handling large dataset (120K+ rows) efficiently
* Data cleaning and standardization issues
* Maintaining performance in Power BI
* Designing a clean and user-friendly dashboard

---

## ✅ Solutions Implemented

* Used BigQuery for efficient data processing and transformation
* Cleaned and structured data using SQL queries
* Designed a multi-page dashboard to avoid clutter
* Applied a consistent dark theme for better user experience
